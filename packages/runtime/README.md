# Kookaburra Runtime

Kookaburra core abstractions

## Connector
Dependency chaining

## Invocation
Validates Input then execute Operation

## IO
Operation input/output

## Locator
Runtime addressing

## Operation
- Retrieves current state
- Executes algorithm
- Persists changed state

## Runtime
Exposes named Invocations
- Creates IO
- Invokes Invocation

## Schema
Input and State validation
